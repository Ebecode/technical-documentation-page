JavaScript Basics

JavaScript Data Types

The JavaScript programming language has seven basic data types:

  • Boolean: true or false.
  • null: Indicates no value and is case sensitive.
  • Undefined: Indicates a property whose value has not been defined.
  • String: indicates one or more characters between brackets, representing a text value. Example: "bananas"
  • Number: indicates an integer or floating point numerical value, like 342 or -4.67.
  • Object: A sequence of key-value pairs between curly brackets. (Arrays are a special kind of object)
  • Symbol (new in EcmaScript 2015): A data type with unique and inmutable instances.
