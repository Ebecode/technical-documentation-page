JavaScript Basics

JavaScript Data Types

The JavaScript programming language has seven basic data types:

  • Boolean: true or false.
  • null: Indicates no value and is case sensitive.
  • Undefined: Indicates a property whose value has not been defined.
  • String: indicates one or more characters between brackets, representing a text value. Example: "bananas"
  • Number: indicates an integer or floating point numerical value, like 342 or -4.67.
  • Object: A sequence of key-value pairs between curly brackets. (Arrays are a special kind of object)
  • Symbol (new in EcmaScript 2015): A data type with unique and inmutable instances.

Operators

Operators allow us to create interactions between values within programming expressions. There are several types of operators, but this documentation will focus on the most basic types.

Unary operators modify the values placed beside or within them. Binary operators create an interaction between the values placed on both sides of them, outputing a third, distinct value.

One example of a unary operator is the typeof operator. Also included are the increment ++ and decrement -- operators.

Binary operators are better known outside programming. These include the following:

  • Arithmetic operators: +, -, *, /, %.
  • Compound operators: +=, -=, *=, /=.
  • Comparison operators: <, >, &let;, &get;.
  • Logical operators: && (logical AND), || (logical OR).
  • Equality operators: == (equal), != (non equal), === (strictly equal), !== (strictly non equal).

Variables

Variables are expressions that temporarily store a value until they are assigned a different value. They conform a very important part of the JavaScript syntax.

Declaring a variable means creating it or calling it into existence. This is how you declare a variable: var x; In thiis example, x can be any name you like as long as it starts with a lowercase letter, has no empty spaces or is one of a few reserved words within the JavaScript language.

Assigning a value means giving a variable a value to store. This is how you assign a value to a variable if it has already been declared: x = 45;. If a variable has not been declared yet, you can declare it and assign a value withiin the same expression: var y = 3;. You can assign a different value to a variable any time you like. For example: y = "Happy Birthday". Now the value will be "Happy Birthday instead of 3.

Functions

Functions are sets of programming expressions you can reuse. Instead of typing several lines of code needed to get a certain result over and over again, these can be stored within a function. And every time you need to use those lines of code, you just have to call or invoke that function.

This is how you create a basic function: function anyName() {//first expression
//second expression
//third expression}

And this is how you invoke a function: anyName();

Most functions have a set of parameters, these are placeholders that go between the parenthesis beside the name of the function. The values you replace those parameters with are called arguments. The function will then use those arguments to perform its calculations. Also, functions may have a return value. This is the part of the the function process that you wish to output as a result of calling the function.

Example: function randInt(min,max) { return ((Math.random()-0.5) * (max - min)) + min; }

The previous example takes in a set of aminimum vand a maximum desired value as arguments. Using those arguments, the function will return the result of a mathematical calculation that should generate a random number between those two arguments.
